# Gameboy Tetris

This project is ready to be hosted for free with GitHub Pages.

## Publish on GitHub Pages

1. Create a new **public** repository on GitHub.
2. Upload `Gameboy.html`, `index.html`, and `.nojekyll` to the repository root.
3. In the repository, open **Settings** -> **Pages**.
4. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
5. Select your main branch and the `/(root)` folder, then click **Save**.
6. Wait a minute or two for GitHub Pages to finish publishing.

Your site URL will be:

`https://YOUR_GITHUB_USERNAME.github.io/YOUR_REPOSITORY_NAME/`

## Update the game later

Replace or edit `Gameboy.html`, push the change to GitHub, and GitHub Pages will republish the site automatically.
